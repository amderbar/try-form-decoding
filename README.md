# try-form-decoding
Purescriptで"フォームデコーディング"を試してみたもの

## Installation

1. このリポジトリをcloneする
2. ディレクトリを移動して以下のコマンドを実行
    ```
    npm install
    npx bower install
    ```

## Run Sample

```
npm test
```
